# 光线追踪
计算机图形学期末作业

## 约定

+ Trace 光线追踪渲染函数 每次渲染一个场景
+ 每个场景包括 屏幕宽高 物体和光线的数组
+ 主函数中，首先设定一个场景，然后为场景添加物体和光源，然后开始渲染，得到图片
+ 使用智能指针 实现派生类在同一个向量中
